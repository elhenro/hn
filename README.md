# hn

## a bash command shortcut runner

### requirements

### optional requirements
* system monitors
** gtop
** htop
** top
** bmon
** netmon

* theme changer
** wallpapers
** wal color scheme generator script

### installation

* clone
* `npm i`
* `echo "alias hn='node path/to/hn/index.js';" >> .bash_aliases`
* `. ~/.bash_aliases`
* `hn`

### usage

* `hn v`
* `hn help`
* `hn gs`
* `hn`
